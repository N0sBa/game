# ⚡ Quick Start Guide

## Test Locally (5 minutes)

```bash
# 1. Install dependencies
npm install

# 2. Start server
npm start

# 3. Open browser
# Go to: http://localhost:3000
```

## Deploy Online (10 minutes)

### Option 1: Railway (Easiest)

1. Go to https://railway.app
2. Sign up with GitHub
3. Click "New Project" → "Deploy from GitHub repo"
4. Select this repository
5. Wait 2-3 minutes
6. Copy the URL and share with friends!

### Option 2: Render

1. Go to https://render.com
2. Sign up with GitHub
3. Click "New +" → "Web Service"
4. Connect this repository
5. Set:
   - Build: `npm install`
   - Start: `node server.js`
6. Deploy and share URL!

## Share with Friends

Once deployed, you'll get a URL like:
- `https://your-game.railway.app` or
- `https://your-game.onrender.com`

**Just share this link!** Friends can play immediately in any browser.
